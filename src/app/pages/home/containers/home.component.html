Vindo(a) esse projeto é uma exemplificação de reatividade e boas praticas envolvendo estrutura de projeto, rxjs, conceitos de estados, observables e uso do angular material temos as seguintes páginas:

  • home
  • usuarios
  • carrinho de itens
  • pessoas
  • lista de tarefas

nesse projeto ha varios conceitos em conjunto como, router, behaviour subject + observables, signals input e output state do navigation em um service a parte para usar de varios lugares root, api service global e a de cada pagina, refatoracao de codigo, ChangeDetectionStrategy.OnPush
